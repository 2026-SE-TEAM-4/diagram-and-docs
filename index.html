04 / POOL

서버 풀 명세서

백엔드가 1분 주기로 PULL해 가는 모니터링 대상 서버 풀(server-pool)의 기능·API 명세. `/health`·`/metrics` 계약과 cpu/mem/gpu/net 메트릭, 백엔드 ServerMetric 추적성.

→
05 / UI

프론트엔드 시안 (KKM)

역할별(STU·MGR·ADM) 화면 시안. 서버 현황·예약·승인·Quota·운영(AIOps 파이프라인)·서버 상태(90일 가동 이력)까지, 백엔드 API·서버풀 데이터가 화면에 어떻게 노출되는지. 정보구조(IA)는 `frontend-prototype-kkm/frontend-ia.drawio`.
